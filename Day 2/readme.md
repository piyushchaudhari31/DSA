## 🧠 Programs Included

### 1. Print N Numbers
Print numbers from 1 to N.

### 2. Sum of Numbers
Calculate the sum from N to 1.

### 3. Reverse Number Sequence
Print numbers in reverse order.

### 4. Strong Number
Check if a number is a Strong Number  
👉 Example: `145 = 1! + 4! + 5!`

### 5. Automorphic Number
Check if a number is Automorphic  
👉 Example: `46² = 2116` (ends with 46)
